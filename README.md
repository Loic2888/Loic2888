​

​

### What’s inside?

# 🌐 Check out my project: [Web Analyser](https://github.com/Loic2888/Lens)
 — [Site](https://web-analyser-client.onrender.com/)

Web Analyser is a full-stack B2B intelligence platform that takes any company URL as input and returns structured, actionable insights in seconds — powered by AI and built for revenue teams.

### What's inside?

**AI-powered company analysis:** enter any website URL and get an instant intelligence brief covering company identity, tech stack, go-to-market signals, and a scored B2B SaaS fit assessment — all generated by Claude or Gemini with deterministic, reproducible results.

**Smart tech stack detection:** the scraping engine extracts meta tags, script sources, link hrefs, and nav signals from the target site, feeding them to an LLM that maps the full frontend, analytics, marketing, and infrastructure landscape without any third-party enrichment API.

**Batch analysis & export:** upload up to 50 URLs at once for asynchronous processing with live progress tracking. Every result is exportable as a formatted PDF report or raw JSON, and saved to a paginated history per account.

**Secure user accounts:** full authentication system with JWT, bcrypt password hashing, email and password management, and cascading account deletion — all with rate limiting and security headers production-ready out of the box.

**Smooth developer experience:** entirely self-contained, built with a Node/Express backend, React 19 + Vite frontend, PostgreSQL on Supabase, and packaged with Docker for instant local deployment with a single command.

Whether you are a sales team qualifying prospects before a call, a founder mapping the competitive landscape, or a developer exploring LLM-powered data extraction pipelines, Web Analyser provides a practical, production-ready tool to turn any website into structured intelligence instantly.
